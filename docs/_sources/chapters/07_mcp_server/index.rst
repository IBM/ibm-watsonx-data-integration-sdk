MCP Server
==========

The ``ibm_watsonx_data_integration_mcp`` package provides an MCP (Model Context Protocol) server that enables AI assistants to interact with IBM watsonx.data integration workflows directly, without needing to write SDK code.

This chapter covers everything you need to know about setting up and using the MCP server with your AI assistant.
