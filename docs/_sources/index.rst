ibm-watsonx-data-integration SDK for Python
===========================================

The ``ibm-watsonx-data-integration`` SDK for Python enables developers and data engineers to programmatically interact with the watsonx.data integration service, IBM's next-generation data integration platform built to support modern, hybrid-cloud data pipelines.

With this SDK, users can automate and manage Flow lifecycles such as creating, configuring, starting, stopping, and monitoring Flows directly in code.

The ``ibm-watsonx-data-integration`` SDK is supported on Python 3.10+.
